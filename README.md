# Simple Calculator Application

This is a basic calculator application built with .NET Windows Forms. It supports fundamental arithmetic operations and provides essential functionalities for a smooth user experience.

## Features

- **Arithmetic Operations**: Addition, subtraction, multiplication, and division.
- **Clear and Delete**: Reset the display or delete the last input.
- **Sign Toggle**: Switch between positive and negative numbers.
- **Decimal Support**: Handle calculations with decimal numbers.
- **Keyboard Input**: Enter numbers and operations using the keyboard.

## Technologies Used

- **.NET Framework**
- **C#**
- **Windows Forms**
- **Visual Studio**

## Conclusion

The Simple Calculator Application offers a user-friendly interface for basic arithmetic tasks, making it a great example of a simple .NET Windows Forms project.
